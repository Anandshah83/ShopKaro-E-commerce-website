# ShopKaro Frontend

## Setup

```bash
cd frontend
npm install
npm start
```

The frontend runs at http://localhost:3000
The backend must be running at http://localhost:5000

## Features

- 🏠 Beautiful homepage with hero, categories, featured products
- 🛍️ Product listing with search, filter by category, sort, pagination
- 📦 Product detail with image, reviews, add to cart
- 🔐 Auth: Login / Signup with JWT
- 🛒 Cart with quantity management, persistent across sessions
- 💳 Checkout with Razorpay payment integration
- 📋 My Orders page
- 👤 Profile page
- 🔧 Admin Panel: Users, Products, Orders, Categories management

## Stack

- React 18 + React Router v6
- Axios for API calls
- react-hot-toast for notifications
- CSS-in-JS (inline styles)
